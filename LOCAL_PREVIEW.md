# 本地预览

- 开发预览固定地址：`http://127.0.0.1:5175/`
- 生产一体化预览固定地址：`http://127.0.0.1:8787/`
- 本项目 API 固定保留：`http://127.0.0.1:8787/`
- 双击 `local-preview.bat` 启动开发预览。
- 命令启动开发预览：`.\local-preview.ps1`
- 命令启动生产一体化预览：`.\local-preview.ps1 -Mode preview`
- 生产一体化预览会先构建前端，再由 Express 同时托管 `dist` 和 API。
- 端口被其他项目占用时，脚本会拒绝启动并显示占用 PID 和命令。
